# Modelo 3GPP TR 38.901

**Versión:** 2026-05-08

## 1. Introduccion
3GPP TR 38.901 es una referencia moderna para escenarios de redes moviles 5G y LTE avanzadas. Define modelos dependientes del entorno, la geometria, la frecuencia y la presencia de trayectorias con o sin linea de vista.

## 2. Escenarios Tipicos
- UMa: Urban Macro
- UMi: Urban Micro
- RMa: Rural Macro
- InH: Indoor Hotspot

## 3. Estructura Conceptual
El modelo suele distinguir entre condiciones LOS y NLOS y combina terminos de perdida de trayecto con variabilidad estadistica.

## 4. Forma General
La perdida total puede expresarse como:

$$
L_{total} = L_{path} + X_{shadow} + X_{small\_scale}
$$

donde los terminos aleatorios o semialeatorios modelan sombreado y variacion adicional.

## 5. Entradas
- Frecuencia
- Distancia 2D y/o 3D
- Altura de antenas
- Escenario seleccionado
- Estado LOS/NLOS

## 6. Salidas
- Perdida de trayecto por celda
- Potencia recibida
- Mapas de cobertura diferenciados por escenario

## 7. Ventajas
- Alto grado de realismo para sistemas modernos.
- Parametrizacion por escenario.
- Apto para analisis LTE/5G y comparacion entre entornos.

## 8. Limitaciones
- Mas complejo que un modelo empirico clasico.
- La parametrizacion debe mantenerse consistente con el escenario elegido.

## 9. Uso en el Sistema
Se utiliza cuando el usuario necesita un comportamiento mas cercano a redes moviles modernas y desea distinguir entre escenarios y condiciones LOS/NLOS.
