# Modelo COST-231

**Versión:** 2026-05-08

## 1. Introduccion
COST-231 extiende la familia Hata para frecuencias mas altas y escenarios urbanos densos, incluyendo una correccion especifica para entornos metropolitanos.

## 2. Rango de Uso
- Frecuencias mayores que el rango clasico de Hata.
- Distancias tipicas de cobertura macro.
- Escenarios urbanos, suburbanos y metropolitanos.

## 3. Ecuacion General
Una forma habitual del modelo es:

$$
L_{COST231}(dB) = 46.3 + 33.9\log_{10}(f) - 13.82\log_{10}(h_b) - a(h_m)
+ \left[44.9 - 6.55\log_{10}(h_b)\right]\log_{10}(d) + C_m
$$

donde $C_m$ representa la correccion del entorno, especialmente para areas metropolitanas.

## 4. Entradas
- Frecuencia
- Distancia
- Altura de antena base
- Altura de antena movil
- Tipo de entorno

## 5. Salidas
- Perdida de propagacion ajustada
- Potencia recibida estimada
- Mapa de cobertura resultante

## 6. Diferencias respecto a Hata
- Extiende el uso a frecuencias mas elevadas.
- Introduce una correccion adicional para entornos metropolitanos.
- Conserva el caracter empirico y logaritmico del comportamiento de atenuacion.

## 7. Uso en el Sistema
Se selecciona cuando el escenario de simulacion requiere un modelo empirico compatible con frecuencias mas altas y alta densidad urbana.

## 8. Limitaciones
- Sigue siendo un modelo de promediacion y no representa completamente la variabilidad espacial fina.
- Requiere coherencia estricta de unidades.
